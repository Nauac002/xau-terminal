XAU TERMINAL · GUIA DE DEPLOY

GitHub + Render.com + UptimeRobot · Gratuito · 15 minutos
Resultado final: O seu terminal fica em https://xau-terminal.onrender.com (ou URL similar), acessível de qualquer browser, qualquer país, sem proxies, sem VPN, com dados em tempo real do Yahoo Finance e FRED.

1CRIAR CONTA NO GITHUB (2 min)

O GitHub é onde o código vai ficar. O Render vai buscar o código de lá automaticamente.

  1. Aceda a github.com e crie uma conta gratuita (se ainda não tiver)
  2. Clique em New repository
  3. Nome: xau-terminal
  4. Deixe Public e clique Create repository

2FAZER UPLOAD DOS FICHEIROS (3 min)

Dentro do repositório criado, clique em "uploading an existing file" e faça upload dos 4 ficheiros que descarregou:

xau-terminal/ ├── server.js ← servidor Node.js (busca dados) ├── package.json ← dependências npm ├── render.yaml ← configuração do Render ├── .gitignore ← ficheiros a ignorar └── public/ └── index.html ← o terminal visual
Atenção: O ficheiro index.html deve ficar dentro de uma pasta chamada public. No GitHub, clique "Create new file", escreva public/index.html como nome — isso cria a pasta automaticamente.

3CRIAR CONTA NO RENDER (2 min)

O Render vai executar o servidor Node.js gratuitamente.

  1. Aceda a render.com
  2. Clique Sign Up e escolha Continue with GitHub — liga as duas contas
  3. Autorize o Render a aceder ao GitHub

4CRIAR O WEB SERVICE (3 min)

  1. No dashboard do Render, clique New + → Web Service
  2. Selecione o repositório xau-terminal
  3. Configure assim:
Name: xau-terminal Region: Oregon (US West) ou Frankfurt Branch: main Runtime: Node Build Command: npm install Start Command: node server.js Plan: Free
  1. Clique Create Web Service
  2. Aguarde 2-3 minutos — o Render vai instalar as dependências e iniciar o servidor
  3. Quando aparecer Live, o seu terminal está online!
URL do terminal: https://xau-terminal.onrender.com
Verificar se funciona: https://xau-terminal.onrender.com/health → deve retornar {"status":"ok"}

5MANTER O SERVIDOR ACORDADO — UPTIMEROBOT (5 min)

O free tier do Render dorme após 15 minutos sem visitas. O UptimeRobot pinga o servidor a cada 5 minutos para o manter acordado — grátis.

  1. Aceda a uptimerobot.com e crie conta gratuita
  2. Clique Add New Monitor
  3. Configure:
Monitor Type: HTTP(s) Friendly Name: XAU Terminal Keep-Alive URL: https://xau-terminal.onrender.com/health Monitoring Interval: Every 5 minutes
  1. Clique Create Monitor
Pronto! O UptimeRobot vai pingar /health a cada 5 minutos. O Render vai pensar que está sempre em uso e nunca vai adormecer. Custo total: €0.

6TESTAR O TERMINAL

  1. Abra https://xau-terminal.onrender.com em qualquer browser
  2. Clique BUSCAR DADOS AGORA
  3. Todos os chips devem ficar verdes — sem proxies, sem bloqueios
Se algum chip falhar: O Yahoo Finance ocasionalmente bloqueia IPs de servidores cloud. Nesse caso, o servidor tenta automaticamente o endpoint v2 do Yahoo. Se persistir, tente de novo em 2 minutos — é rate limiting temporário, não um erro permanente.
